# Sorting Machine

## 설치

`npm install`

## 시작

`npm run start`

## 기능 구현

**타이머**

-   [x] 타이머 구현(재활용 가능한 컴포넌트)

**입력**

-   [x] 사용자의 입력 결과 출력
-   [x] 입력 데이터 예외 처리

**시작**

-   [x] 버튼 클릭시 소팅 구현
-   [x] 오름차순은 바로 출력 되고 3초 후에 내림차순 출력

**결과**

-   [x] 오름차순 정렬 구현
-   [x] 내림차순 정렬 구현
-   [x] 소팅 내장 메소드X, 직접 구현O
